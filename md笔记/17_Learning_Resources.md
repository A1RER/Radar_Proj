## 📄 论文资源

### 综述论文（必读）
```
1. Deep Learning for Radar: A Survey
   IEEE Signal Processing Magazine, 2019
   下载：IEEE Xplore
   评价：★★★★★ 全面了解雷达深度学习

2. A Survey of SLAM Algorithms
   IEEE Transactions on Robotics, 2015
   下载：IEEE Xplore
   评价：★★★★★ SLAM综述经典

3. Recent Advances in GNSS/INS Integration
   GPS Solutions, 2018
   评价：★★★★☆ 组合导航最新进展
```

```
ISAR成像:
- Global Range Alignment for ISAR (2001)
- Autofocus Technique for ISAR (1996)

- SAR Automatic Target Recognition Using CNNs (2016)
- Deep Learning for SAR Image Classification (2018)

定位导航:
- The Kalman Filter for Position Estimation (1960原文)

```

### 论文搜索技巧
```
Google Scholar: https://scholar.google.com
- 搜索关键词："ISAR imaging" "radar target recognition"
- 按引用量排序，找经典文献

知网: https://www.cnki.net
- 搜索中文论文
- 下载学位论文（硕博论文很有参考价值）

IEEE Xplore: https://ieeexplore.ieee.org
- 搜索最新会议论文
- 重点关注：ICRA, IROS, RadarConf
```

## 🎯 学习建议

### 高效学习方法
```
1. 项目驱动学习
   - 不要只看教程，要做项目
   - 边做边学效率最高

2. 理论+实践结合
   - 看完教材就写代码
   - 写完代码回头看理论

3. 主动输出
   - 写技术博客
   - 整理学习笔记
   - 给别人讲一遍

4. GitHub记录
   - 所有项目上传GitHub
   - 代码要整洁，有README
```

### 时间管理
```
周中（每天2小时）：
- 1.5小时：项目实践
- 0.5小时：理论学习

周末（每天5小时）：
- 3小时：项目实践
- 1小时：论文阅读
- 1小时：整理笔记
```

### 避坑指南
```
❌ 只收藏不学习
❌ 追求完美不动手
❌ 看太多教程不实践
❌ 代码写得乱七八糟

✅ 收藏后立即学习
✅ 先做出来再优化
✅ 看一个教程就实践
✅ 代码规范，能展示
```
